# cours-intro-edition-num

Repo pour déployer les slides du cours d'introduction à l'édition numérique, L3 Lettres, parcours édition/création, Univesrité de Rouen, janvier 2026

## Cheatsheet des commandes

`source venv/bin/activate`

`pip install nbconvert`

`jupyter nbconvert cours1.ipynb   --to slides   --reveal-prefix ./reveal.js`
